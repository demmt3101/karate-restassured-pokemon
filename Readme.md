## 🎯 Objetivo

Implementar pruebas automatizadas contra **PokeAPI**:

- Base URL: `https://pokeapi.co/api/v2`
- Endpoint principal: `GET /pokemon/{id|name}`

---

## 🧩 Casos mínimos requeridos

Usar (al menos) estos casos:

```json
{
  "cases": [
    { "id": 1, "name": "bulbasaur" },
    { "id": 4, "name": "charmander" },
    { "id": 7, "name": "squirtle" }
  ]
}
